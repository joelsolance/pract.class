# 🗂️ Buscador de Carpetas

Script de Python para buscar carpetas en un disco externo de forma rápida sin tener que navegar manualmente por el explorador de Windows.

## ¿Qué hace?
- Escanea todas las carpetas y subcarpetas de un disco externo
- Permite buscar una carpeta por nombre
- Abre automáticamente la carpeta en el explorador de Windows
- Copia la ruta al portapapeles para usarla en "Guardar como"

## Requisitos
- Python 3.x
- pyperclip

## Instalación
pip install pyperclip


## Uso
1. Conecta tu disco externo
2. Ejecuta el script
3. Escribe la letra de tu disco (ejemplo: E)
4. Escribe el nombre de la carpeta que buscas
5. El script abre la carpeta y copia la ruta automáticamente
